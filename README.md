# ComprehensiveCalculator
为了完成安卓期末大作业，我模仿小米计算器实现了一款多功能计算器，可以进行数学运算、汇率转换、亲戚关系转换、长度转换、面积转换、体积转换、温度转换、速度转换、时间转换和重量转换。
